# Legend of Lampião | Lenda de Lampião

## Jogo desenvolvido no 3º estágio da disciplina de Programação III
## Desenvolvido no semestre 2014.1 para obtenção de nota


- O jogo consiste em um *lampião* matando zumbie;
- Jogo de plataforma;
- Todo o motor e jogo foi desenvolvido em java, sem uso de nenhuma biblioteca de terceiro;

Jogo desenvolvido por mim, Marcelo S. de Azevedo e Girlian Antônio dos Santos, colegas de classe.
